# End-to-End Pizza Sales Analytics Project

## Overview

This project demonstrates an end-to-end Business Intelligence workflow, transforming raw pizza sales data into actionable insights through data warehousing, SQL analytics, and interactive Power BI reporting.

The solution includes data modeling, analytical SQL queries, KPI development, and a sales dashboard designed to support data-driven decision-making.

## Objectives

- Build a structured data warehouse for pizza sales data.
- Perform exploratory and analytical SQL analysis.
- Generate business insights related to revenue, sales performance, customer ordering behavior, and seasonality.
- Develop an interactive Power BI dashboard for stakeholders.

## Tech Stack

- SQL Server
- Data Warehouse (Star Schema)
- SQL Analytics & Reporting
- Power BI
- DAX

## Project Workflow

1. Data ingestion and preparation
2. Data warehouse design and implementation
3. Fact and dimension modeling
4. Exploratory Data Analysis (EDA)
5. Business KPI development
6. Power BI dashboard creation
7. Insight generation and reporting

## Key Metrics

- Total Revenue
- Total Orders
- Average Order Value (AOV)
- Average Pizzas per Order
- Daily Sales Trends
- Monthly Seasonality Analysis
- Category Performance
- Pizza Size Analysis
- Top & Bottom Performing Products
